**Amazon_Sales_dashboad-powerbi**

**1. Amazon Sales Dashboard : Amazon Sales Insights Dashboard**

This is an interactive dashboard where we talks about total amount and shipping cost, category distribution, top 10 product and top countries according to category


---

**2. Description :**

The Amazon Sales Dashboard is a visually engaging and analytically robust reporting tool. Based on the dataset analysis, the project highlights include: 91.83M in sales revenue.**Electronics** is the largest category (16.85%), followed by Sports, Books, Home & Kitchen, Toys & Games, and Clothing. Profit margins across all categories remain relatively low.74.69% of orders have been successfully **Delivered**, 15% are **Shipped**, and a very low percentage of products are returned.**Credit Cards** dominate with 35% of total transactions, followed by Debit Cards (20.02%) and UPI (15.07%).

3.Tech-Stack :

The dashboard was built using the following technologies:

* 📊 Power Bi Desktop : The main data visualization platform for dashboard creation.
